Q4. what is the diff betn throw and throws keyword?
Throw : used inside a method to explicityly throw an exception.
ex- if (age < 18) {
    throw new ArithmeticException("Not eligible to vote");
}

Throws : used in the method declaration to indicate that method may throw one or more checked exceptions.
ex: void readFile() throws IOException, SQLException {
    // risky code
}

combine example-
void validate(int age) throws Exception {
    if (age < 18) {
        throw new Exception("Invalid age");
    }
}
